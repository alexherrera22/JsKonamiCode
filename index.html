jQuery Konami Code.

Many old school Konami videogames used the next secuence to cheat on the game, like start with 30 lives in Contra:

Up, Up, Down, Down, Left, Right, Left, Right, B, A

Today, some websites use it to do something special. You can see some of them in Konami Code Sites and with this jQuery plugin you can easily do your own.
